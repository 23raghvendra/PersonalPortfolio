## Overview
A modern weather application that provides real-time weather data and 5-day forecasts for any city worldwide. Features a clean, responsive design with smooth animations and dark mode support.

## Problem
Most weather apps are cluttered with ads and unnecessary features, making it hard to quickly check the weather. Users need a fast, clean interface that shows essential weather information at a glance.

## Solution
Built a lightweight, vanilla JavaScript weather app that focuses on speed and user experience. Uses the Open-Meteo API for accurate weather data without requiring API keys, making it completely free to use.

## Key Features
- **Real-time Weather Data** - Get current conditions for any city instantly
- **5-Day Forecast** - View upcoming weather predictions with detailed information
- **Geolocation Support** - Automatically detect and display weather for your location
- **Temperature Units** - Seamlessly switch between Celsius and Fahrenheit
- **Dark Mode** - Eye-friendly dark theme with smooth transitions
- **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
- **Beautiful Animations** - Smooth transitions and visual effects for better UX
- **No Dependencies** - Pure vanilla JavaScript with no frameworks or libraries

## Technical Highlights
- Clean, semantic HTML5 structure
- Custom CSS animations and transitions
- Modern ES6+ JavaScript features
- Efficient API calls with error handling
- LocalStorage for theme persistence
- Responsive design using CSS Grid and Flexbox
